Format for EOS_PST.dat:
energy-desnity (GeV/fm^3)     pressure (GeV/fm^3)     entropy-density (fm^-3)     temperature (GeV)

Format for EOS_Mu.dat:
energy-desnity (GeV/fm^3)	Mu_i (GeV)

Format for EOS_particletable.dat:
first row: number of stable particles
start from second row:
first column: IColMu - 1,   IColMu is the column number in EOS_Mu.dat
second column: correspond particle Monte-Carlo number in pdg.dat
for the comments: they specified the particle name, particle life time(fm), and whether to be considered as stable particle
